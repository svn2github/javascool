Tutoriel 7 : Conversion Binaire -> Decimal




Objectif :


Le but de ce tutoriel est d'écrire un programme qui permet de convertir un nombre codé en base 2 (nombre binaire) en nombre codé en base dix (nombre décimal).


Notes :



Pour réaliser ce tutoriel vous pouvez utiliser les fonctions présente dans la boîte à outils dont le fonctionnement est rappelé ici. En particulier les fonctions mathématique.

Pour le travail demandé, on impose que le nombre binaire soit écrit sur 8 bit.


Vous trouverez ici un rappel sur les tableaux .


Pour rappel l'algorithme est le suivant :

Recommencer de i=0 à i=7

Lire bit(i)

somme = somme + bit(i) * 2(8 - i) 

 Afficher « En décimal, ce nombre vaut », somme  



Travaille demandé :


créer un nouveau fichier de type « jvs » que vous pouvez nommé par exemple : « Tuto7.jvs ».

Écrivez le programme correspondant à l'algorithme de conversion d'un nombre binaire en décimal. La variable associée au nombre binaire sera du type tableau.
