Utiliser l'«educlet» mettre-le-nom.

Description de l'applete.

Cette applete permet d'enregistrer ses propres sons.

Utilisation comme outil.

L'interface capture les sons via le microphone de l'ordinateur. Une représentation temporelle du signal capté est dessiné en tant réel. Il s'agit d'un signal stéréo, d'où les deux évolutions au court du temps.
  • Pour débuter et stopper l'enregistrement, taper 'r'.
  • Pour sauver la capture du signal, taper 's'. L'enregistrement sera alors stocké dans dans le dossier "out". Vous pouvez le renommer au besoin.

Utilisation dans une activité.

Cette applete peut être utilisée en préambule de BoiteAMusique, afin de constituer sa bibliothèque de grains sonores avant composition.
