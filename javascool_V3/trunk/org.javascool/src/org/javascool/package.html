Ce package Java implémente les mécanismes JavaScool de proglets destinés à s'initier à la programmation à partir du lycée (niveau ``L0´´).

Il permet bien évidemment d'utiliser toute l'API Java.

Les classes utilisées par l'«élève» sont documentées en français. Les classes utilitaires utilisées pour créer/adapter des proglets dans l'environnement Java sont documentées an anglais. Un wiki permet de partager des contenus et une forge de co-développer.
