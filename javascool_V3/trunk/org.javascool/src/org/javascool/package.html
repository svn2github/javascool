org.javascool standalone interfaces

  • Usage: java org.javascool.Main [activity [file-name]]: Starts a JavaScool (V3) "activity" which result is to be stored in a "file-name".
  • Usage: java org.javascool.MainV2 (edit|run) [proglet-name [file-name]]: Used to edit a program or run a proglet with JavaScool V2.
  • Usage: java org.javascool.Pml input-file [output-file]: Checks the well-formedness of a file by mirroring its structure in a normalized PML format, the file extension defines the format.
  • Usage: java org.javascool.Hml input-file [output-file]: Checks the well-formedness of a file by mirroring its structure in a normalized HML format, the file extension defines the format.
  • Usage: java org.javascool.ProgletApplet proglet: Runs the demo of proglet.
  • Usage: java org.javascool.HtmlDisplay location: Show a HTML3.2 page.
  • Usage: java org.javascool.LinkCheck [-recursive] location: Checks links in the HTML text at the given HTTP location, and optionnaly recurse in sub-directories.
