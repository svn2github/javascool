This is the JavaScool core Java and Xml bundle.
Français/French

Ce package Java implémente les mécanismes JavaScool des «proglets» destinées à s'initier à la programmation à partir du niveau du lycée (niveau ``L0´´).

Il permet bien évidemment d'utiliser toute l'API Java.

Les documents utilisés sont écrits en Hml (c'est à dire en syntaxe XML ou en Pml avec des constructions prédéfinies pour simplifier l'écriture. Ils peuvent aussi être écrits sur le wiki.

Les classes utilisées par l'«élève» sont documentées en français. Les classes utilitaires utilisées pour créer/adapter des proglets sont documentées an anglais. Un wiki permet de partager des contenus et une forge de co-développer cet ensemble.

Anglais/English

This Java package implements the core of JavaScool of «proglets» to be used in order to learn programming, starting at the high-school level.

It indeed allows to use all the Java API.

Documents are to be written using the Hml constructs (using the XML or Pml syntax), with predefined constructs allowing to simplify the writing. They can also be written on the wiki.

Classes used by students are documented in French. Core classes Classes used to create/adapt proglets are documented in English. A wiki (in French) allows to share information and a forge to co-develop this bundle.

XSLT translation rules
hml2htm This XSLT translates HML code to HTML 4 output.
htm2hml This XSLT translates HTNML to HML constructs.
hdoc2htm This XSLT translates the proglet's HML documentation to HTML3.2.

org.javascool standalone interfaces

  • Usage: java org.javascool.Main [activity [file-name]]: Starts a JavaScool (V3) "activity" which result is to be stored in a "file-name".
  • Usage: java org.javascool.Jvs2Java [-reformat] [-compile] input-file: Translates Jvs files to Java files. Optionally reformat the Jvs code and/or compile the java classes.
  • Usage: java org.javascool.Pml input-file [output-file]: Checks the well-formedness of a file by mirroring its structure in a normalized PML format, the file extension defines the format.
  • Usage: java org.javascool.Hml input-file [output-file]: Checks the well-formedness of a file by mirroring its structure in a normalized HML format, the file extension defines the format.
  • Usage: java org.javascool.ProgletApplet proglet: Runs the demo of proglet.
  • Usage: java org.javascool.HtmlDisplay location: Show a HTML3.2 page.
  • Usage: java org.javascool.LinkCheck [-recursive] location: Checks links in the HTML text at the given HTTP location, and optionnaly recurse in sub-directories.
