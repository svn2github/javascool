Utilisation d'un éditeur externe.

L'utilisateur peut bien évidemment préférer son éditeur externe favori à l'éditeur intégré à la plateforme JavaScool.
  • Il suffit de définir la variable standard d'environnement $EDITOR
    (par exemple "export EDITOR=nedit" ou tout autre nom de programme d'édition déjà installé).
  • Lors du lancement du programme, il faut taper CTRL+E dans la fenêtre d'édition pour que celle-ci se transforme en fenêtre d'affichage et que l'éditeur externe se lance.
  • Sous Windows ou si la variable $EDITOR n'est pas définie, le mécanisme fonctionne. Mais c'est à l'utilisateur de lancer manuellement l'éditeur externe sur le fichier sauvegardé lors du vérouillage de l'éditeur interne.
