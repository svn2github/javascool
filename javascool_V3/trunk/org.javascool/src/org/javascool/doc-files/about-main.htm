Aide à l'utilisation de l'interface.

Utiliser l'interface se fait en trois temps: (1) choisir son activité, (2) éditer le fichier de travail en suivant ce qui est proposé dans la documentation, (3) Compiler/tester le résultat. Voici comment son disposés tous les éléments:
tandis que le tutoriel «Hello World» permet de s'initier avec la mécanique de cet interface.

    Quelques Astuces.

  • Sauver les fichiers : Attention à bien sauvegarder les fichiers dans un dossier de l'utilisateur (sur le "bureau" ou dans le dossier "documents"), sinon le système risque de se plaindre d'une tentative d'écriture illicite.

Table des raccourcis.

Editeur de texte ou donnée.

Control O (open) Ouvrir un fichier pour l'éditer
Control S (save) Sauvegarder le contenu édité dans un fichier
Control E Vérouille/Dévérouille l'éditeur interne, pour éditer le fichier avec un éditeur en externe
Control L Recolorie le code en cours d'édition
Control H (help) Montre la page d'aide
Control Q (quit) Quitter le logiciel
Control I (info) Affiche les informations d'attestation de JavaScool, dont le numéro de révision
Control C (copy) Copier la portion de texte ou de donnée sélectionnée
Control X (cut) Couper la portion de texte ou de donnée sélectionnée
Control V (paste) Coller la portion de texte ou de donnée précédemment copiée ou coupée

Editeur de texte.

Control Z (undo) Annuler la dernière commande d'édition
Control Y (redo) Restaurer la dernière commande d'édition

Navigateur de documentation.

Control HOME (home) Revenir à la page initiale
Control <- (backward) Revenir une page en arrière
Control -> (forward) Avancer une page en avant
