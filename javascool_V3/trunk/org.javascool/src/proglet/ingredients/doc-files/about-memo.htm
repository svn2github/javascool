Memorandum des constructions et instructions

Adjectifs : type de données utilisées.

  • String Chaîne de caractères (ex: String nom = "Bod";).
  • int Entier positif ou négatif, de environ -2 à +2 milliards.
  • double Nombre décimal, comme sur une calculette (ex: double pi = 3.1416;).

    // Les nombres décimaux se notent avec un "." point et non une virgule !

    // Les nombres en notation scientifique (par exemple 6.023 10 23) se note avec un "e" (pour exposant) : ici 6.023e23
  • boolean Valeur vraie ou fausse, utilisée dans les tests.

Verbes: fonctions utilisées.

Macros générales.

  • void echo(String string) Affiche une chaîne de caractères sur la console1.
  • boolean equal(String string1, String string2) Compare deux chaînes de caratères et renvoie true si elles sont égales.
  • double pow(double x, double y) Calcul x à la puissance y (soit x y). Note : Math.pow est synonyme de pow.
  • double random() Renvoie un nombre aléatoire uniformément distribué entre 0 et 1. Note : Math.random est synonyme de random.
  • int random(int min, int max) Renvoie un nombre aléatoire entier uniformément distribué entre le min et le max (cette valeur incluse).
  • void sleep(int duree) Arrête l'exécution du programme pendant une durée fixée donnée en milli-secondes.

Fonctions de la proglet Konsol.

  • void clear() Efface tout le contenu la fenêtre de sortie.
  • void println(String string) Ecrit une chaînes de caratères dans la fenêtre de sortie.
  • String readString() Lit une chaîne de caratères dans la fenêtre d'entrée (ex: String nom = readString();).
  • int readInteger() Lit un nombre entier dans la fenêtre d'entrée (ex: int x = readInteger();).
  • double readDouble() Lit un nombre décimal dans la fenêtre d'entrée (ex: double x = readDouble();).
  • boolean readBoolean() Lit une valeur vraie (true, vrai, oui, 1) ou fausse (false, faux, non, 0) dans la fenêtre d'entrée.
(note: readInt est synonyme de readInteger, readFloat est synonyme de readDouble).

Grammaire: constructions utilisées ici.

Bloc de séquences d'instructions.

Un bloc se note entre { } et englobe un ensemble de déclarations et d'instructions séparées par un ;.

Opérateurs.

= Affectation d'une valeur à une variable (ex: x = 0;)
|| && ! Opérateur logique OU, ET et NON
== != <= < >= > Opérateur égalité, différence et comparaison numérique
+ - * / % Addition, soustraction, multiplication, division, reste (appelé aussi modulo).

Instruction conditionnelle.

Elle se note
if (expressionBooleenne) { BlocD'InstructionsSiVrai } else { BlocD'InstructionsSiFaux }
le else et le bloc qui suit peut être omis (voir l'activité sur les instruction conditionnelle pour plus de détail).

Boucle.

Elle se note
while (expressionBooleenne) { BlocD'Instructions }
(voir l'activité sur les boucles pour plus de détail).

Déclaration de fonction.

Elle se note
TypeDuResultat NomDeLaFonction (TypeParametre1 NomParametre1, ..., TypeParametreN NomParametreN) { BlocD'Instructions }
(voir l'activité sur les fonctions pour plus de détail).

Programme principal.

Il se note void main{ } avec les instructions définies entre { }.
tandis que le langage Java a beaucoup d'autres instructions et opérateurs.

Remarques :

1La différence entre echo et println est echo affiche la chaîne de caractères dans la console, donc marche avec toutes les proglets, tandis que println est dédiée à la fenêtre de sortie de la «proglet» Konseol.
