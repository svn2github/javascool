La «proglet» Konsol.

Cette proglet[.] permet de définir des entrées/sorties textuelles au clavier comme pour une console[.].
  • On peut donc dans la fenêtre input:

    // Lire une chaine de caractère avec la fonction:

        String chaine = readString();

    // Lire un nombre entier avec la fonction:

        int entier = readInteger();

    // Lire un nombre décimal avec la fonction:

        double decimal = readDouble();
    où les constructions proposées ici utilisent des [.].
  • On peut aussi écrire dans la fenêtre output:

    // Ecrire une chaine de caractère avec la fonction:

        println(chaine);
    cette chaine pouvant être une constante, ou se voir ajouter des variables numériques comme dans les exemples ci dessous:
        println("Hello world");
        println("Valeur = " + valeur);
    où ici valeur est une variable numérique entière ou décimale.
Un exemple d'utilisation: en appuyant sur la touche [ Demo] le programme suivant:

// Efface le contenu de la fenêtre de sortie

clear();

// Salue l'utilisateur et lit son nom

println("Bonjour, qui es-tu ?");
String nom = readString();

// Lui demande alors son âge

println("Echanté "+nom+" ! Quel âge as-tu ?");
int age = readInteger();

// ../..

s'exécute pour permettre d'expérimenter le fonctionnement de cette proglet.

Remarques.

  1. Cette proglet est un [.], programmée en Java et [.] pour sa libre redistribution, l'accès [.], et permettre des développements dérivés.
  2. Une console ou « terminal alphanumérique » correspond à une fenêtre où l'utilisateur peut dialoguer avec un programme en entrant des données au clavier et en lisant les chaines de caractères imprimées par le programme.
