Exercices et applications
La 20taine d'exercices activités du document de ressources publié par ÉduSCOL a été instanciées en Java'sCool : les "soluces" (aide syntaxique) sont proposées ici (nous contacter pour l'accès). Avec le parcours pour s'initier à la programmation, elles permettent de couvrir le programme de 2nd en algorithmique.
Quelques autres applications, exercices et activités :
  • Comprendre le principe de la recherche dichotomique.
  • Programmer quelques calculs géométriques.
  • Découvrir comment marche un convertisseur analogique-numérique.
  • Comprendre quelques opérations de manipulation d'images.

  2. Chaque proglet[.] est un code source libre, programmée en Java et documenté pour sa libre redistribution, avec un accès au code source, et permettre des développements dérivés.
    Un enseignant qui connait un peu de programmation peut facilement développer ces propre proglet's, non sans vous inviter à nous demander (dans la mesure de notre disponibilité) de le faire à partir de votre idée, de vous y aider, et nous de faire partager les vôtres, etc.
Une proglet est une petite applette qui permet de s'initier à la programmation de manière ludique, puisque les petits programmes créés animent un élément visuel ou sonore, à la fois pour mieux comprendre ces objets numériques et pour offrir une démarche expérimentale concrète, vers la compréhension des notions abstraites qui forment le fondement des sciences de l'information. Ces proglets sont des exemples de «gamelet» (c'est à dire de grains pédagogiques interactifs pour un apprentissage ludique d'une notion), telle que définie par fuscia.
Ces documents ne sont pas conçus pour apprendre la programmation en "autodidacte" ... mais si vous le souhaitez et si vous pouvez dialoguer avec quelqu'un de votre entourage qui sait programmer (afin de ne pas être bloqué sur des détails de syntaxe ou d'implémentation) alors oui : n'hésitez pas à suivre le parcours pour s'initier à la programmation : vous allez vite épater ceux qui pensent être les seuls à savoir programmer !
