La «proglet» Synthe
Cette proglet1 permet de générer une séquence de notes et d'en regarder le spectre de fréquence.
Pour définir son propre son il faut utiliser la construction Java'sCool, par exemple:
@tone: sqr(t) + noi(t)
où @tone: est une « macro-instruction », suivie, sur la même ligne d'une expression de la variable t qui définit la forme du son, tandis que les fonctions suivantes sont prédéfinies:

// Sinusoide

double sns(t)

// Signal carré

double sqr(t)

// Signal triangulaire

double tri(t)

// Souffle aléatoire

double noi(t)
Pour écouter le son ou la séquence de notes il faut utiliser la fonction:
synthePlay();
La visualisation du son s'effectue en montrant:
  • Les amplitudes des fréquences du son (tracé en rouge, l'unité est de décibel, renormalisé pour offrir le meilleur tracé).
    Les fréquences sont tracées entre le La sous-grave (octave 0 à 27.5 Hz, à peine audible) et le La sur-aigu (octave 9 à 6400 Hz), tandis que le milieu de l'échelle est le La du diapason (octave 3 à 440 Hz).
  • Le début du signal (tracé en jaune), permet de voir sa forme et son aspect temporel. Les premières 11ms (0.011 secondes) sont tracées.
Pour définir une séquence de notes il faut utiliser la fonction:
syntheSet("séquence-de-note");
où la syntaxe de "séquence-de-note" est définie dans la documentation de la proglet.
Un exemple d'utilisation: en appuyant sur la touche [ Demo], les différents sons:
@tone: cos(t)
synthePlay();
puis
synthePlay();
@tone: 0.5 * sqr(t)
puis
synthePlay();
@tone: 0.8 * tri(t) + 0.2 * noi(t)
puis
synthePlay();
@tone: 0.3 * sqr(t/2) * sin(t) + 0.3 * sin(2 * t) + 0.3 * tri(3 * t)
sont présentés, puis les 1ères notes de quelque chose qui ressemble à la «Lettre à Elise» de L.v.Beethoven est jouée et son spectre affiché et ensuite
. . à vous de jouer !

Remarques :

1Cette proglet est un code source libre, programmée en Java et documentée pour sa libre redistribution, l'accès au code source, et permettre des développements dérivés.
