Cette proglet1 permet de simuler de programmer toutes sortes d'applet en utilisant la syntaxe Java ordinaire.
On peut utiliser toute l'API Java et l'API JavaScool.
On utilisera en particulier les classes de l'API Swing, dont on trouve facilement des exemples illustrés et un tutoriel complet
Plus précisément la fonction getSwingPane renvoie le panneau Java dans lequel l'interface graphique sera défini, par exemple avec la construction:

JLayeredPane pane = getSwingPane();

qui donne à la variable pane la valeur du JLayeredPane où se feront les dessins.
Ne pas oublier d'ajouter les imports des classes Java utilisées par exemple en écrivant:

import javax.swing.*;

au début du fichier.
Un exemple d'utilisation est proposé en démonstration, à partir d'un code Jvs réalisé avec deux lycéens du Lycée International de Valbonne.
