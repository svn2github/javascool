Tutoriel 4 : Les boucles




Objectif :


Le but de ce tutoriel est de vous familiariser avec les boucles et de vous en apprendre le fonctionnement.


Notes :


Voici un exemple de programme qui affiche trois fois la phrase « HelloWorld » :

Exemple1{

   void main(){

      for(int i=0; i < 3 ; i++){

            println("HelloWolrd");

      }

   }

}

Le fichier correspondant : Exemple1.jvs

Exemple1{

   void main(){

      int i=0;
      while(i < 3){
            println("HelloWolrd");
            i++;
      }

   }

}

Le fichier correspondant : Exemple2.jvs

Travaille demandé :


Dans le premier exemple expliquer ce que signifie l'expression : "i++"


Si on remplace dans le premier exemple "i<3" par "i <=2" le programme fait-il la même chose ?


les deux exemples réalisent-ils la même chose ? expliquez pourquoi ?


Que se passerait-il si on ne met pas l'instruction « i++ » dans le deuxième exemple ?
