Java's Cool Guide Utilisateur : Introduction



Java's Cool est une application programmé en JAVA 1.6, basé sur le noyau Eclipse 3.3, compatible Windows Xp & Vista, Mac os 10.4 & 10.5 et Linux. Elle a pour but d'apprendre les fondements de la programmation informatique à ceux qui le désir. Dans cette finalité, il est fourni, en plus du logiciel, un ensemble de tutoriaux (exercices et solutions) et un guide complet pour bien prendre en main l'interface utilisateur. Il est important de faire attention à l'extension de sauvegarde du fichier qui peut être ".jvs" ou ".java". Dans le premier cas l'utilisateur fera du java's cool qui est un macro-langage au dessus de java. Il pourra utiliser la boîte à outils. Dans le second cas, l'éditeur se comportera comme un pur éditeur java sans aucune modification du code lors de la compilation. La boîte à outils ne fonctionnera pas.
