Introduction

L'onglet Principal permet de créer des instructions de saisie (champ "entrées") ou d'affichage (champ "sorties") :

Deux boutons peuvent être utilisés :

  • Le bouton Nouveau créera un nouveau programme gérant non seulement les entrées-sorties, mais aussi la déclaration de variables.
  • Le bouton Insérer complétera un programme existant, en ajoutant uniquement les instructions de saisie et d'affichage.

Bouton Insérer : attention !

Le bouton Insérer ayant d'autres usages (sous-programmes notamment), il n'ajoutera les instructions de saisie et d'affichage que si la condition suivante est réalisée :

Cette condition est réalisée si vous cliquez sur le bouton Effacer pour chaque nouvel exemple à traiter.
