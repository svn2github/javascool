Exemple 3 : saut à skis

Dans un concours de saut à skis, chaque skieur est noté par 5 juges. Sa note finale est obtenue en sommant les 5 notes, puis en soustrayant à cette somme la note minimale et la note maximale...

Etape 1 : gestion d'un skieur (version élémentaire avec tableau)

Le problème peut en grande partie être résolu avec PluriAlgo :

  • Fixez le langage de développement (Javascool ou autre langage).
  • Complétez l'onglet Principal (indication), puis cliquez sur le bouton Nouveau.
  • Complétez l'onglet Boucles (indication), puis cliquez sur le bouton Insérer.
  • Adaptez le programme (indication).

Etape 2 : gestion d'un skieur (avec enregistrements ou classes)

Il est possible de reformuler le programme de l'étape 1 en introduisant un type Skieur regroupant son nom et ses 5 notes :

  • Recopiez si nécessaire ce programme (consulter) dans l'éditeur (avec un copier-coller).
  • Complétez l'onglet Principal (indication), puis cliquez sur le bouton Reformuler.
  • Adaptez le programme (indication pour les enregistrements ou les classes).

L'étape 2 n'est applicable qu'à 4 langages : Javascool, Visual Basic, Python et Carmetal. Pour les 2 derniers, vous devez typer les variables lors de la reformulation (indication).

Etape 3 : gestion de plusieurs skieurs

Il est possible (en Javascool uniquement) de passer de un à plusieurs skieurs, en introduisant un tableau de skieurs :

  • Recopiez si nécessaire le programme de l'étape 2 (enregistrements ou classes) dans l'éditeur (avec un copier-coller).
  • Complétez l'onglet Boucles (indication), puis cliquez sur le bouton Transformer.
  • Complétez la boîte de dialogue qui apparaît alors (indication).

Pédagogiquement, il est souhaitable de compléter le code obtenu, par exemple avec un algorithme de tri pour classer les skieurs : voir solution (enregistrements ou classes). Mais, cela sera sans l'aide de PluriAlgo qui vient d'atteindre son seuil d'incompétence !

    Remarque : seuil pas tout à fait atteint puisque vous pouvez poursuivre l'exemple en traduisant la solution dans un autre langage...
