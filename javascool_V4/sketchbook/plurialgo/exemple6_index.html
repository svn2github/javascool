Exemple 4 : saisie d'une enquête

Introduction

Dans une enquête de santé, diverses informations sont demandées aux personnes interrogées : nom, sexe, taille, poids...

Afin de privilégier l'étude des options de saisie proposées par PluriAlgo, le traitement de l'enquête sera limité au calcul de l'indice de masse corporelle : poids divisé par le carré de la taille.

Formulaire

Pour construire un formulaire de saisie d'une personne (consulter), vous pouvez procéder ainsi :

  • Complétez l'onglet Principal (indication), puis cliquez sur le bouton Nouveau.
  • Complétez les boîtes de dialogue qui apparaîssent alors pour la variable "nom" (indication) et pour la variable "sexe" (indication).
  • Complétez le programme avec le calcul de l'IMC (voir résultat).

Pour construire un formulaire de saisie de plusieurs personnes (consulter), vous pouvez procéder ainsi :

  • Complétez l'onglet Principal (indication), puis cliquez sur le bouton Nouveau.
  • Complétez le programme avec le calcul de l'IMC (voir résultat).

La démarche suivie en Javascool peut être reproduite avec Java, Javascript, Php, Python ou R. Par contre, PluriAlgo ne gère pas la traduction d'un programme avec formulaire (ou fichier texte ou base de données).
