La «proglet» javaProg.

Cette proglet permet de simuler la programmation de toutes sortes d'applets en utilisant la syntaxe Java ordinaire.
On peut utiliser toute l'API Java et l'API JavaScool.
On utilisera en particulier les classes de l'API Swing, dont on trouve facilement des exemples illustrés et un tutoriel complet.
Voici d;autres liens pour s'initier aux «Swing» (interfaces graphiques) Java:
  • sur developpez.com (B. Wicth)
  • sur le site du zero
Plus précisément la fonction getPane renvoie le panneau Java dans lequel l'interface graphique sera défini, par exemple avec la construction:
JLayeredPane pane = getPane();
qui donne à la variable pane la valeur du JLayeredPane où se feront les dessins.
Ne pas oublier d'ajouter les imports des classes Java utilisées par exemple en écrivant:
import javax.swing.*;
au début du fichier.
