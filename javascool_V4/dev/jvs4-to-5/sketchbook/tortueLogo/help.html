La «proglet» tortueLogo.

Cette proglet permet de simuler la célèbre tortue logo.
Dans ce contexte les instructions à utiliser sont celles de Java'sCool, la tortue évolue dans un carré de salade de taille [512 x 512] tandis que les primitives graphiques usuelles de la tortue logo sont disponibles. La tortue est toujours visible et ne peut sortir du jardin et disparaître de l'écran graphique.
Toutes les instructions disponibles sont documentées ici
Que les nostalgiques de cette ancienne façon d'apprendre à programmer en profite, la tortue est de retour:
Un exemple d'utilisation: lors de la démonstration, la tortue trace de belles courbes de Lissajous en changeant la couleur:

// Initialisation du tracé

clear_all();
pen_up();

// Boucle de tracé

int t = 0; while(t < 9000) {
    set_color((t / 1000) % 10);
    set_position(
        256 + 250 * Math.cos(0.0015 * t),
        256 + 250 * Math.sin(0.0045 * t)
    );
    pen_down();
    t = t + 1;
}
