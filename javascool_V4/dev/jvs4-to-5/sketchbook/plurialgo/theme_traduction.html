Traduction de programmes Javascool

Un programme Javascool peut, sous certaines conditions restrictives, être traduit dans une dizaine d'autres langages :

  1. Téléchargez si nécessaire le programme Javascool (exemple 1, exemple 2).
  2. Dans l'onglet "Principal", fixez le langage de traduction (Python, Visual Basic, Javascript, Php...) et cliquez sur le bouton Traduire.

Il ne faut pas rêver : le mécanisme de traduction ne fonctionnera pas pour n'importe quel programme ! Mais j'ai cherché à ce qu'il s'applique aux situations les plus courantes.


Remarque : le mécanisme de traduction ignorant l'affichage de messages fixes, il est conseillé de mettre les résultats dans des variables (voir restrictions).

Le mécanisme de traduction fonctionne également si l'on part d'un programme Algobox (sous forme d'algorithme ou de fichier), Larp (exemple) ou Visual Basic (exemple 1, exemple 2).
