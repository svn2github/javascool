L'onglet Principal

L'onglet Principal permet de créer automatiquement des squelettes de programmes, à partir d'informations telles que les variables d'entrée ou de sortie et de leur type (réel, texte...) : par exemple (télécharger), vous pouvez calculer un prix total (variable de sortie) à partir d'un nombre d'articles et d'un prix unitaire (variables d'entrée). Il reste alors à cliquer sur le bouton Nouveau, après avoir fixé le langage de programmation : javascool, visual basic (vb), larp, javascript...

Les variables introduites peuvent être soit de type simple (entier, réel, texte, booléen), soit des tableaux à une ou à deux dimensions de type simple. Par exemple, vous pouvez introduire des tableaux (télécharger) dans le but de gérer un bon de commande :
