Tutoriel 2 : "HelloWorld"




Objectif :


Le but de ce tutoriel est de vous apprendre à écrire des programmes qui écrivent des chaînes de caractères, des nombres, .... sur la sortie standard 1.


Notes :


Pour réaliser ce tutoriel vous pouvez utiliser les fonctions présentes dans la boîte à outils dont le fonctionnement est rappelé dans le manuel d'utilisation.

Voici un exemple de programme qui affiche sur la sortie standard 1 la phrase "HelloWorld".


HelloWordl{

   void afficher(){

      println("HelloWorld");

   }

   void main(){

      afficher();

   }
}

Le fichier correspondant : HelloWorld.jvs

Travaille demandé :


Dans le programme qui vous est donné en exemple essayez; de comprendre le rôle de la phrase sur-ligné en bleue.


Créez un nouveau fichier de type « jvs » que vous pouvez nommer par exemple : « Tuto2.jvs ».

Dans ce nouveau fichier créez une fonction « afficherPrenom » qui prend en paramètre une chaine de caractère (le prénom;) et qui affiche sur la sortie standard 1  le prénom en paramètre.

Créez une seconde fonction « afficherNom » qui prend en paramètres une chaine de caractère (le nom) et qui affiche sur la sortie standard 1  le nom en paramètre.

Créez une fonction main


Ensuite créez la fonction «  main »2 qui appelle les deux fonctions définis précédemment pour afficher le prénom et le nom.
















1 La sortie standard est le flux de sortie dans lequel les données sont écrites par le programme. Une manière simple, bien que techniquement inexacte, de concevoir le principe de la sortie standard est de le considérer comme « le périphérique vers lequel le programme envoie les données ». Ainsi, si le périphérique en question est l'écran, « écrire dans la sortie standard » revient à « écrire sur l'écran » ou « afficher à l'écran ».


2Pour rappelle la Signature de la fonction main est : « void main(){...} »
