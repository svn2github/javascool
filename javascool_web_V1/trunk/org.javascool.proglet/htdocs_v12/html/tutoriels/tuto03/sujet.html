Tutoriel 3 : Lecture sur la Console




Objectif :


Le but de ce tutoriel et de vous apprendre à lire différents types de valeurs choisit par l'utilisateur sur l'entrée standard1. .


Notes :


Pour réaliser ce tutoriel vous pouvez utiliser les fonctions présentes dans la boîte à outils dont le fonctionnement est rappelé dans le manuel d'utilisation.

Voici un exemple de code qui lit sur l'entrée standard1 une chaîne de caractère (un élément de type String) , ainsi qu'un nombre et qui affiche les deux éléments lus sur la sortie standard.

Exemple{

   void lireInfo(){

      println("entrez une chaine : ");
      String chaine = readString();
      println("chaine lu : " + chaine);
      println("entrez un nombre : ");
      int nombre = readInt();
      println("nombre lu : " + nombre);

   }

   void main(){
      lireInfo();
   }

}

Le fichier correspondant : Exemple.jvs

Travaille demandé :


Pour réaliser les questions suivantes vous pouvez, (il vous est même fortement conseillé) de vous baser sur l'exemple de code fournit.


Créer un nouveau fichier de type « jvs » que vous pouvez nommer par exemple : "Tuto3.jvs".


Créer une fonction "lirePrenom" sans paramètres qui lit sur l'entrée standard1 le prénom d'une personne et qui retourne le prénom lue.

Créer une fonction "lireAge" sans paramètres qui lit sur l'entrée standard1 l'age d'une personne et qui retourne l'âge lue.


Ensuite créer la fonction main 2 qui lance la lecture du prénom et l'âge d'une personne et qui affiche les deux réponses lues.
