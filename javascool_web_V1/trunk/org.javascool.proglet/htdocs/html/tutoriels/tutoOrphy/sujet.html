Tutoriel 4 : Les boucles




Objectif :


Le but de ce tutoriel est de vous familiariser avec la récupération de données sur Orphy GTS 2 par l'intermédaire du code Java's Cool.


Notes :


Voici un exemple de programme qui affiche la valeure d'un thermomètre branché sur l'entrée analogique G d'ORPHY.
On voit bien l'initialisation et la fermeture d'orphy avec les methode demarrer() et arreter(), indispensables.

Exemple1{

   void main(){

      Orphy.demarrer();

      double temperature = Orphy.acquisitionEntree("G", "Thermomètre");

      println("la température est : " + temperature);

      Orphy.arreter();

   }

}

Le fichier correspondant : Exemple1.jvs


Travaille demandé :


Faire un programme qui affiche la valeur de l'entrée G comme dans l'Exemple1, mais cette fois-ci 20 fois, avec un interval de 1 sec(soit 1000 milli-secondes) entre chaque mesure.
Astuces :

  • Il est conseillé d'utiliser une boucle for pour faire le programme.
  • Pour faire un intervalle de 1 sec entre chaque acquisition, utiliser la maccro dort(int millisecondes) accessible par la boîte à outils.

Plus difficile :
Meme exercice qu'avant, mais cette fois, faire que le relevé ne s'arrête que si la température est supérieure à 30°.
Possibilités pour stopper une boucle :

  • faire appel à l'instruction "break;" si la condition est respéctée
  • utiliser une variable booléenne comme condition d'arret que l'on mettra à false si la condition est remplie à un moment donné
