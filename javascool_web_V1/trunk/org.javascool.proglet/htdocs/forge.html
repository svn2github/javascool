SourceForge de l'INRIA



Accès au code source

Le projet est sous SVN, il peut etre récupéré par la commande :
     svn checkout svn://scm.gforge.inria.fr/svn/javascool.

Vous devez au préalable vous inscrire sur la SourceForge de l'INRIA.



Le projet est installé sur la gforge INRIA, vous pouvez y acceder à l'adresse : https://gforge.inria.fr/projects/javascool.
