Tutoriel 10 : Lecture de Fichiers




Objectif :


Le but de ce tutoriel est de vous familiariser avec la lecture et l'écriture de fichiers.


Notes :


Pour réaliser ce tutoriel vous pouvez utiliser les fonctions présente dans la boîte à outils dont le fonctionnement est rappelé dans le manuel d'utilisation. En particulier les fonctions d'ouverture et de lecture d'un fichier.


//lechemin du fichier a ouvrir en lecture
String path = "C:\\......\\fichier.txt">
openReadFile(path); //ouverture du fichier en lecture

.....
//fermeture du fichier ouvert en lecture
closeReadFile();

Voici un exemple de programme qui ouvre un fichier en écriture :

//lechemin du fichier a ouvrir en ecriture
String path = "C:\\......\\fichier.txt">
openWriteFile(path); //ouverture du fichier en ecriture

.....
//fermeture du fichier ouvert en lecture
closeWriteFile();

Le fichier texte dont vous allez vous servir pour ce Tutoriel : bit.txt


Travaille demandé :


En vous basant sur les exemples donnés précédemment, Ecrivez un programme qui ouvre en lecture le fichier texte fournit et qui lit tous les Bits de ce fichier (par groupe de 4 bits) et qui les codes en Hexadécimal.
Faites afficher sur la sortie Standart le résultat de la conversion pour chaque nombre.


Modifier votre programme écrit précedemment de manière à écrire le résultat de la conversion de chaque nombre, non pas sur la sortie standart mais dans un fichier texte (fichier ayant pour extension ".txt").


Réalisez le même exercie mais cette fois en prennant pour la conversion en Hexadecimal les bits par groupe de 6.



Comparez les deux fichiers texte résultants de la conversion en hexadecimal que remarquez-vous ?
